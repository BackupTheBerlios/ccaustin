Content Management System

The CMS package provides the data model for how content is stored and retrieved from the content repository. To get started, use the RepositoryFactory to provide the appropriate information to obtain a reference to a Repository implementation, which may be either local or remote. A Repository contains Sites, which contains Folders, which contains other <Folders and Files. The format of the Files in the Repository is opaque, and it is therefore up to external application interfaces to define how the content is treated based on its content type and the content itself.
