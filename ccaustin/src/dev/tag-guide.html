Tag Reference Guide

This document provides a reference guide for all content tags available for creating content.

Tag Overview

These tags are for marking up standard content for a web page. You will find that these tags will work within every page on the website. The basic rules for content within the web site is that all content within a page must be surrounded with a <content> tagset, and all text should be surrounded by the <p> tag. Some pages may define custom tags to further describe the content for the page, or store more structured content. It is up to the page writer's discretion to document these tags at the top of each content page. Refer to each page's documentation at the top for instructions on composing the page.

When in doubt about a tag's specific use, refer to an existing document within the website that uses the tag or consult another content writer or web servant for more information.

Tag Summary

The default tags for a web page are grouped into the following areas. Each tag is linked to more detailed information found later in this document.

Basic Tags
These tags are used quite often, and comprise the essential tags necessary to compose just about any page.
content Contains all content for a page. A page must have 1 content tag and it must contain all other tags for the page.
p Defines a paragraph of text.
br Starts a new line inside a paragraph
img Inserts an image into the page
link Inserts a link to another page or within the document
heading Defines a major section heading or a heading for the page
subheading Defines a minor section heading or a subheading within the page

 

Tables/Data Structures
These tags allow data to be grouped into tables or lists for better display and management.
table Defines a new table.
row Adds a row to the table
label Adds a table cell label and its contents
data Adds a table cell and its contents to the row
list Defines a new list
item Adds an item to the list

 

Text Control
Tags which control the look of text within a paragraph.
note Marks a portion of text as a notice to the user
emphasis Marks a portion of text as emphasized (italic)
verse Marks a portion of text as a Bible verse
subtext Marks a portion of text as fine print
sidetext Marks a portion of text as a notice (this tag may go away).

 

Special Purpose
Tags that have specific purposes and are not used often but made available for advanced page writers.
redirect A tag for telling the user's browser to go to another page. Useful for redirecting the user from the default page for a web site to the actual home page.

 

Basic Tags

<content> Content Tag
The content tag opens at the beginning of a page's XML contents, and ends at the end. Only things within the content tag are processed by a page and put into the resulting HTML page.
 
Attributes:
title

the title of the page, placed into the titlebar of the browser when viewing
 
Body:
The body of this tag consists of all other page tags in this document. Some pages may also include new tags specific to the page. For information on those tags, refer to the documentation at the top of the xml document.
 
Example: 
<content title="Heart-to-Heart"> 
  <!-- Page tags go inside here --> 
</content>

<p> Paragraph Tag
Groups a paragraph of text together. All text to be shown on a page should be wrapped with this tag.
 
Attributes:
align

left, right, center alignment
 
Body:
The text to display and other page tags to alter the look of the text.
 
Example: 
<p>Each spring the Heart-to-Heart ministry pairs women together so they can
develop supportive, God-centered friendships.</p>
