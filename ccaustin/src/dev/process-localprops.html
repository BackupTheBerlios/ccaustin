local.properties

This file is used to store the user id and password for the FTP server, and other important information. There is an example.local.properties in CVS that may be copied and named local.properites, should the need to publish from a specific workstation or server be required. Storing ids and passwords in this file enables our publish script to automatically perform important tasks (publishing our HTML to the production web server) without actually storing ids and passwords in CVS that someone may be able to break into in the future.
