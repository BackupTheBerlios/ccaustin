XSL Template

An XSL template, also called an XSL Stylesheet, defines the rules by which an XML document's contents get converted into something else. For the purposes of the Calvary Web Site, this means converting XML (the content) with XSL (the layout) to produce HTML (what everyone sees). In general, XSL is like a basic scripting language, as it defines things called templates that match a particular XML tag, and operate on that tag.

The following is an example of an XSL template:

<?xml version="1.0" encoding="ISO-8859-1"?>
<xsl:stylesheet version="1.0"
xmlns:xsl="http://www.w3.org/1999/XSL/Transform"><xsl:template match="/">
 <html>
 <body>
   <h2>My CD Collection</h2>
   <table border="1">
     <tr bgcolor="#9acd32">
       <th>Title</th>
       <th>Artist</th>
     </tr>
     <tr>
       <td><xsl:value-of select="title"/></td>
       <td><xsl:value-of select="artist"/></td>
     </tr>
   </table>
 </body>
 </html>
</xsl:template></xsl:stylesheet>

As you can see, its simply an XML document that defines a template, or rule, and a set of HTML to produce for that rule. Within each rule the XML data can be referenced so that when combined, they can produce an HTML page.
