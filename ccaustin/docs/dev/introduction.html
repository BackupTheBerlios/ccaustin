Introduction

The Calvary Web Site is part of the Calvary Internet Ministry and provides the overall Calvary Austin web presence. The web site is critical for the following groups:

  • Church seekers - those looking to move into the Austin area or Austinites locating a new church
  • Calvary members - those who call Calvary their church home and wish to obtain more information or view a live broadcast
  • The lost - those searching the web and locating our content and what we believe (video, audio, text)

The mission of the Calvary Internet Ministry mirrors the Calvary Chapel Austin mission statement:

  • Win
    • Reach the lost
    • Introduce the church and the Calvary Chapel Movement
    • Public discussion forums for apologetics, cults, etc.
  • Equip
    • Communicate events/news with the body
    • Present transcripts and audio/video archives of services
  • Send
    • Virtual missionaries to the web

Under God's direction, we plan to accomplish these things using the following methods:

  • A fresh web site with continually changing content
  • Moderate use of cutting-edge technologies (server-side dynamic content, client-side animation such as Flash)
  • Strong marketing using strategically crafted HTML for better search engine location, registration with local city directories (Citysearch) and national church directories
  • Interactive presentations on the Gospel
  • Continued use of RealAudio and RealVideo technologies for archived and live broadcasts of Sun, Wed, and other weekly services
  • Participation areas for discussions with those who have questions about Christianity, the Calvary Chapel Movement, and other relevant topics

The Internet Ministry can use people with all kinds of talent, including:

  • HTML scripters (Javascript, CSS, XML/XSL a plus)
  • Web centeric graphic artists (static, Flash, etc.)
  • Java programmers (servlets and JSP for dynamic web pages)
  • Network engineers (server backups, network configuration)
  • Anyone willing to keep content up to date - NO COMPUTER EXPERIENCE REQUIRED!
